Multiple Choice Quiz Shuffler

Paste any multiple-choice test below. The tool detects numbered questions like 1., 2., 3. and options like a), b), c), then shuffles both the questions and the options.

It does not require an answer key. It simply reorders the test based on structure.

Example:

1. Το κύριο όργανο του κυκλοφορικού συστήματος είναι:
a) η καρδιά.
b) τα αγγεία.
c) το αίμα.
